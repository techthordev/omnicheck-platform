# OmniCheck Platform 🚀

An enterprise-grade monitoring dashboard for IT Support Engineers. This platform provides real-time visibility into infrastructure health, container status, and system logs.

## 🛠 Tech Stack
- **Backend:** Java 25 (Temurin/Mandrel), Spring Boot 4.0
- **Frontend:** Angular 21 (Signals-first)
- **Database:** PostgreSQL 18
- **Infrastructure:** Podman (Native on Fedora Linux)

## 📂 Project Structure
- [/backend](./backend): Spring Boot REST API
- [/frontend](./frontend): Angular Single Page Application
- [/infrastructure](./infrastructure): Podman orchestration (Dev/Prod)
- [/db/migration](./db/migration): Flyway SQL scripts
- [/docs](./docs): Technical Documentation
